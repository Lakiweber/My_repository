1. Регистрируемся на сайте 
- Заходим на https://github.com/join?source=login
- При регистрации запоминаем login и pass
- На указанную почту при регистрации приходит письмо, проверяем почту и подтверждаем с помощью ссылки
- В личном кабинете заполняем профиль

2. Устанавливаем Desktop версию для совместной работы
- Скачиваем программу https://desktop.github.com/
- Инсталлируем программу

3. Вводим данные в программу
- запускаем программу "GitHub Desktop"
- Выбираем меню File -> Options
- На вкладке Accounts в строчке GitHub.com нажимаем "Sign in"
- Вводим свои login  и pass, при правильно выполненных операциях вы войдете в программу под своим аккаунтом
- Во вкладке Git заполните поля Name и Email это необходимо для контакта с Вами

4. Для подключения к проекту 
- Отправить письмо на почту lakiweber@gmail.com с указанием своего login
- В ответ вы получите письмо с ссылкой для приглашения в проект, пройдите по ссылке и примите приглашение

5. Добавления репозитория
- запускаем программу "GitHub Desktop"
- Выбираем меню File -> Clone repository
- Переходим на вкладку URl и вставляем в это поле следующую строку без кавычек "https://github.com/Lakiweber/Transcorp.git"
- В строке Local path указываем папку на вашем ПК где будут хранится файлы, жмем кнопку Clone

  Вы подключены к проекту!
